# Changelog

本项目遵循轻量维护节奏。版本号对应 `manifest.json` 中的扩展版本。

## 0.2.1

- 支持粘贴抖音 / 小红书分享链接或整段分享文案后采集。
- 打开分享短链后等待跳转到真实视频页面再提取媒体地址。

## 0.2.0

- 增加多 provider 转写支持。
- 支持阿里百炼 DashScope、火山引擎 / 豆包语音、腾讯云 ASR、OpenAI Audio、Deepgram、AssemblyAI。
- 支持自定义转写 API 模板。
- 增加更完整的 provider 配置说明和平台限制说明。

## 0.1.0

- 初始版本。
- 支持在抖音和小红书页面采集视频地址。
- 支持调用用户自己的转写 API。
- 支持导出 Markdown、复制 Markdown 和通过 Obsidian URI 导入 Obsidian。
